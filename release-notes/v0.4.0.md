# HomeOS v0.4.0

Release date: 2026-05-01

## Highlights

- Adds a Cosmos bypass-warn audit reader that mirrors recognized Cosmos UI Docker mutations into `/var/log/homeos-audit.jsonl` with `verdict=BYPASS`.
- Adds `homeos audit cosmos-events` to inspect Cosmos-origin audit entries.
- Documents the Cosmos v0.4 audit path in `docs/AI-GATE.md`, `docs/DAY2.md`, and the README.
- Fixes Debian Installer handling for the reserved final username `admin` by installing with a temporary `diadmin` account and renaming it during `late_command`.
- Refreshes pinned GitHub tool SHAs.

- Shell syntax checks passed for changed Bash scripts.
- YAML parsing passed for changed Ansible/YAML files.
- Cosmos audit reader passed a fake-Docker log emission test.
